## 📌 Overview

A-Plus-Hire is a professional NDIS hiring and recruitment platform designed specifically for the disability support sector in Australia.

The platform enables:

* 👨‍⚕️ Support Workers to discover and apply for jobs
* 🏢 NDIS Providers to recruit and manage candidates
* 🧑‍🦽 Participants to connect with qualified support professionals
* 👑 Administrators to manage the entire hiring ecosystem

The application includes role-based dashboards, secure authentication, responsive UI, recruitment workflows, and a scalable backend architecture.
